SchemaBackup
============

Intended as SQL Server backup tool, backup to SVN.

Limitations
-----------

Currently only exports DB schema to a working folder, no SVN upload yet

Software Requirements
---------------------

<ol>
<li>svn.exe is in your <i>$PATH</i> variable</li>
</ol>
